# Game Design Document  

## Fourmula - Kidnapped: Castle Breakout  

Fourmula - Kidnapped: Castle Breakout is een escape room-achtige game waarin spelers puzzels moeten oplossen en kamers verkennen om te ontsnappen uit een mysterieus kasteel. Door items te verzamelen en raadsels op te lossen, vorderen spelers door het verhaal en ontdekken ze geheimen over hun gevangenschap.
