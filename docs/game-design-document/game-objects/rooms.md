De player start in een slaapkamer waarin hij een puzzel moet oplossen of de sleutel moet vinden om te ontsnappen. Als de player opgepakt wordt, start hij weer in deze kamer.

Er is een main room waarin de player andere kamers kan betreden. Bijvoorbeeld een gang met deuren naar de andere rooms.

Uiteindelijk moet hij de poort openen om te ontsnappen 



Een donker kamer waarin de player met een lamp de sleutel moet vinden en door een geheime doorgang naar de volgende kamer moet komen. In de 2de kamer moet de player met een raadsel oplossen van de spook om een item te krijgen om de volgende kamer te betreden.
